# Destinations

  * android components (intents [implicit, explict], activities, fragments, broadcast receivers, intent services),
  * notifications,
  * data binding,
  * permissions,
  * camera,
  * gesture detector,
  * daggerAndroid,
  * retrofit,
  * room persistence storage,
  * rxJava,
  * Glide,
  
and displays all the places we've already visited with my second half ;)
